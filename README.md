## Project Overview

Customer churn occurs when a customer stops using a service. Predicting churn helps businesses retain at-risk customers and reduce revenue loss.  

This project uses the **Telco Customer Churn dataset** to predict churn using machine learning and deep learning models.

---

## Dataset

- **Filename:** `WA_Fn-UseC_-Telco-Customer-Churn.csv`
- **Features:** Customer demographics, account info, and services
- **Target:** `Churn` (`Yes` = churn, `No` = no churn)
